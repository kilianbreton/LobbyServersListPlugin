# LobbyServersList Plugin

### How to configure

Create a json on your server or upload on a website and put the path/link in the setting **Config file**

Json example : 

```json
[
  {
    "Name" : "ServerName",
    "Login" : "ServerLogin",
    "JoinPassword" : "Join password",
    "SpecPassword" : "Spectate password",
    "TitlePack" : "SMStormElite@nadeolabs
  },
  {
    "Name" : "Combo",
    "login" : "ankou_combo",
    "TitlePack" : "SMStormCombo@nadeolabs"
  }
]
```
Optional fields :
- JoinPassword
- SpecPassword
- TitlePack
